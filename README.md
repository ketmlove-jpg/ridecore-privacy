# ridecore-privacy
# RideCore 개인정보 처리방침

RideCore는 사용자 위치 정보를 주행 기록 기능 제공 목적으로만 사용합니다.

앱은 사용자가 라이딩을 시작할 때 위치 데이터를 수집하며,
주행 경로, 속도, 거리 등의 기능을 제공하기 위해 사용됩니다.

수집된 데이터는 외부에 공유되지 않으며,
사용자의 동의 없이 제3자에게 제공되지 않습니다.

사용자는 언제든지 앱을 통해 기록 데이터를 삭제할 수 있습니다.
본 앱은 사용자의 명시적인 동작(라이딩 시작)에 의해 위치 정보 수집이 시작됩니다.


## 개인정보 보관 기간

회사는 수집된 개인정보를 서비스 제공 기간 동안 보관하며, 
회원 탈퇴 시 즉시 삭제하거나 관련 법령에 따라 일정 기간 보관 후 삭제합니다.

단, 다음의 경우에는 법령에 따라 보관됩니다.

- 계약 또는 청약철회 등에 관한 기록: 5년
- 소비자 불만 또는 분쟁처리에 관한 기록: 3년

- ## 위치 정보 이용

본 앱은 위치 기반 서비스를 제공하기 위해 사용자의 위치 정보를 수집합니다.

위치 정보는 라이딩 기록, 경로 분석, 안전 기능 제공을 위해 사용되며,
사용자가 서비스를 이용하는 동안에만 저장됩니다.

사용자는 언제든지 위치 정보 수집을 거부할 수 있으며,
이 경우 일부 기능이 제한될 수 있습니다.

## 개인정보 삭제 방법

사용자는 앱 내 설정 또는 고객센터를 통해 개인정보 삭제를 요청할 수 있으며,
요청 시 지체 없이 삭제됩니다.
